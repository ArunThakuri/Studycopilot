# 🚀 Google AI (Gemini) Setup Guide

## Why Google AI (Gemini)?

✅ **Fast** - Process units in 30-60 seconds (vs 3-5 minutes with Ollama)  
✅ **Free** - Free tier with generous limits  
✅ **No Installation** - Works immediately, no local setup needed  
✅ **High Quality** - Powered by Google's latest Gemini models  
✅ **Vision Capable** - Can process both images and text  

---

## 🎯 Quick Setup (Under 1 Minute!)

### Step 1: Get Your Free API Key

1. Go to **[Google AI Studio](https://aistudio.google.com/app/apikey)**
2. Click **"Create API Key"** (or **"Get API Key"**)
3. Sign in with your Google account if needed
4. Select **"Create API key in new project"** (or use existing project)
5. **Copy the API key** that appears

⚠️ **Important**: Save this key somewhere safe! You won't be able to see it again.

### Step 2: Configure StudyCopilot

1. Open StudyCopilot
2. Click **"AI Settings"** button (gear icon)
3. Select **"Google AI (Gemini)"**
4. Paste your API key
5. Click **"Save"**

✅ **Done!** You're ready to use fast AI processing!

## 🔑 API Key Best Practices

### Security
- ✅ Keep your API key private
- ✅ Don't share it publicly
- ✅ Don't commit it to version control
- ✅ Store it securely (password manager)

### Usage Limits
- Free tier includes:
  - 15 requests per minute
  - 1,500 requests per day
  - 1 million tokens per day

- For StudyCopilot:
  - Creating 1 unit = ~6-7 requests
  - You can create ~200 units per day on free tier
  - Perfect for students and teachers!

### If You Hit Limits
1. Wait a few minutes and try again
2. Or upgrade to paid tier (optional)
3. Or switch to Ollama for unlimited local processing

## 🎓 Example: Creating Your First Unit

### With Gemini (Fast - 30-60 seconds):

1. Click "Create New Unit"
2. Upload textbook images OR paste markdown
3. Enter unit title

5. ⏱️ **Wait 30-60 seconds**

## 🔧 Troubleshooting

### "API Key Invalid" Error
- ✅ Double-check you copied the entire key
- ✅ Make sure there are no extra spaces
- ✅ Try creating a new API key

### "Quota Exceeded" Error
- ✅ You've hit the free tier limit
- ✅ Wait a few minutes and try again
- ✅ Or switch to Ollama

### "Network Error"
- ✅ Check your internet connection
- ✅ Check if firewall is blocking Google AI
- ✅ Try again in a few minutes

### Slow Processing
- ✅ Gemini should be fast (30-60s)
- ✅ If slow, check your internet speed
- ✅ Large images take longer to process

## 📖 What Gemini Processes

### 1. **Image to Text** (Vision)
- Extracts text from textbook images
- Recognizes formulas, diagrams
- Preserves structure and formatting

### 2. **Text Cleaning**
- Removes page numbers, headers
- Fixes spelling errors
- Structures content properly

### 3. **Module Generation**
- **Vocabulary**: Extracts difficult words with Nepali translations
- **Audio Transcript**: Creates spoken lesson script
- **Summary**: Generates key points, formulas, concepts
- **Exercises**: Creates practice questions with solutions
- **Quiz**: Generates multiple choice questions
- **Practice**: Creates mixed practice questions

## 🌟 Tips for Best Results

### For Image Processing:
- ✅ Use clear, high-quality images
- ✅ Make sure text is readable
- ✅ Upload pages in order
- ✅ Avoid blurry or dark images

### For Better Modules:
- ✅ Provide accurate unit titles
- ✅ Include complete content (not just snippets)
- ✅ Use the regenerate button if not satisfied
- ✅ Review and edit AI-generated content

### For Faster Processing:
- ✅ Use markdown files instead of images when possible
- ✅ Process one unit at a time
- ✅ Keep image files reasonably sized
