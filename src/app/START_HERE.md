# 🎓 StudyCopilot - Quick Start Guide

Welcome to StudyCopilot! A modern learning management system with **real AI-powered text extraction** from textbook images.

## 🎯 Two Modes

### Demo Mode (Default)
- Instant example content generation
- No setup required
- Perfect for exploring features

### AI Mode (Recommended)
- **Real text extraction from images** (like OCR)
- Uses Ollama for local AI processing
- Extracts EXACT text from textbook pages
- See **QUICK_START_EXTRACTION.md** for 5-minute setup

## 🚀 Get Started in 3 Steps

### 1️⃣ Create Your Account
- Click anywhere to launch the app
- Click "Sign Up" from the landing page
- Fill in any details (demo mode - no real authentication)

### 2️⃣ Add a Subject
- Click "Add New Subject" button
- Enter a subject name (e.g., "Mathematics", "Science")
- Select a grade level (4-12)
- Choose a color theme
- Click "Create Subject"

### 3️⃣ Create Your First Unit
- Click on your newly created subject
- Click "Create New Unit"
- Enter a descriptive title (e.g., "Introduction to Algebra")
- Optionally upload some images or a markdown file
- Click "Process & Create Unit"
- Wait ~2 seconds for instant demo content generation!

## ✨ What You'll Get

Each unit automatically includes:

- 📄 **Unit Text** - Well-formatted educational content
- 🎧 **Audio Lesson** - Transcript for narration
- 📚 **Vocabulary** - 8 words with Nepali translations
- 📝 **Summary** - Key definitions, formulas, and concepts
- ✏️ **Exercises** - 4 varied practice problems
- 🎯 **Interactive Quiz** - 10 multiple-choice questions
- 💡 **Practice Questions** - 4 challenging problems

## 💡 Pro Tips

## 🎭 Demo Mode vs AI Mode

### Demo Mode (No Setup)
- ✅ All features are fully functional
- ✅ Content generates instantly
- ✅ No setup or API keys required
- ⚠️ Uses example content (not real extraction)
- ⚠️ Data doesn't persist (refresh = reset)

### AI Mode (Real Extraction)
- ✅ Extracts EXACT text from textbook images
- ✅ No hallucinations or added content
- ✅ Local processing (privacy guaranteed)
- ✅ Edit extracted content in markdown editor
- ⚠️ Requires Ollama installation (5 minutes)
- ⚠️ Needs vision model (~5GB download)

**To enable AI Mode:**
1. Read **QUICK_START_EXTRACTION.md**
2. Install Ollama
3. Pull a vision model
4. Start extracting!
