## 🐛 Troubleshooting

### Issue: Still getting low quality output
**Solution**: Make sure you're using Gemini, not Ollama

### Issue: Unit Text still has exercises
**Solution**: Check console logs, may need to regenerate

### Issue: Less than 20 quiz questions
**Solution**: Content may be too short, or model couldn't generate 25

### Issue: Practice module not showing answers
**Solution**: This is a UI issue, not prompt issue

### Issue: Parsing errors
**Solution**: Check if using latest code, clear cache

## 💡 Tips for Best Results

### Content Quality
1. Upload clear, readable textbook content
2. Use images with good OCR quality
3. Or use well-formatted markdown

### Provider Selection
1. **Use Gemini for production** (high quality)
2. Use Ollama for testing only
3. Gemini handles complex prompts better

### Module Generation
1. Generate modules sequentially (already implemented)
2. Wait for each to complete
3. Check quality after generation
4. Regenerate if needed
