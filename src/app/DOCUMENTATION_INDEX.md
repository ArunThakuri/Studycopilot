## 📱 Quick Reference Cards

**2-Minute Summary:**
- System extracts EXACT text from images
- No additions, no elaborations
- Like OCR / copy machine
- Test: Image text = Output text

**Common Questions:**

Q: Why is output different from image?  
A: Model might be adding content. Try llava model.

Q: How do I know if it's working?  
A: Output should match image exactly, word-for-word.

Q: Which model is best?  
A: llava or llama3.2-vision:11b for pure extraction.

Q: Can I edit the extracted text?  
A: Yes! Use the markdown editor after extraction.
